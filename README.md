# Photobooth


## Requirements

 * Sofortbild app for Mac http://www.sofortbildapp.com/
 * PHP + Imagick (this dependency will be removed in future versions)
 * ImageMagick


## Installation

Run the ```sh make``` command to create all the required directories.
